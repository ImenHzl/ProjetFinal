23 """
24 Récupère les données météorologiques actuelles pour une ville donnée.
25  
26 Args:

29  
30 Returns:
31 dict: Les données météorologiques actuelles pour la ville spécifiée.
32 """
33 # URL de l'API pour les données météorologiques actuelles
34 url = f"http://api.weatherapi.com/v1/current.json?key={api_key}&q={city}"
35  
36 # Faire une requête GET pour obtenir les données
37 response = requests.get(url)
38  
39 # Vérifier si la requête a réussi
40 if response.status_code == 200:
41 # Convertir la réponse en format JSON
42 data = response.json()

53 """Compare les conditions météorologiques actuelles pour plusieurs villes.

55 Args:
